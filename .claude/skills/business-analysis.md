# 事業構造分析マニュアル

## 分析の目的

対象企業が「何で稼いでいるか」「なぜ稼げるのか」「どんなリスクがあるか」を構造的に理解する。
この分析は全ての投資判断（A/B/C）の基盤となる。

## 分析手順

### 1. 事業セグメントの構造把握

1. 決算短信・有価証券報告書からセグメント情報を取得
2. 各セグメントについて以下を整理:
   - セグメント名と事業概要
   - 売上高と全体に占める割合（%）
   - 営業利益（またはセグメント利益）と全体に占める割合（%）
   - 利益率
3. **単一セグメントが売上の80%以上を占める場合**: そのセグメント内の製品・サービス別、地域別、顧客別などの次のレイヤーまでブレイクダウンする

### 2. PPM（プロダクト・ポートフォリオ・マトリクス）分析

各セグメントを以下の4象限に分類する:

| | 高い市場成長率 | 低い市場成長率 |
|---|---|---|
| **高い市場シェア** | 花形（Star）: 成長投資が必要だが将来の稼ぎ頭 | 金のなる木（Cash Cow）: 安定キャッシュ源 |
| **低い市場シェア** | 問題児（Question Mark）: 投資判断が必要 | 負け犬（Dog）: 撤退・縮小候補 |

### 3. 外部環境分析

主要事業が影響を受ける外部要因を特定する:
- **市場全体の動向**: 属する市場の規模・成長率・成熟度
- **法規制の変化**: 新法・規制緩和・規制強化の影響
- **政治・政策動向**: 国策テーマとの関連（防衛、半導体、脱炭素、インバウンド等）
- **社会変化**: 人口動態、働き方の変化、消費者行動の変化
- **技術変化**: AI、DX、新技術の破壊的影響
- **為替・原材料**: 為替感応度、主要原材料の価格動向

### 4. 競合分析

- 主要事業における直接競合を3〜5社特定
- 競争優位性の源泉を特定:
  - コスト優位性（規模、技術、立地）
  - 差別化（ブランド、技術、ネットワーク効果）
  - スイッチングコスト（顧客の乗り換え障壁）
  - 参入障壁（免許、特許、設備投資の大きさ）
- 競合との定量比較: 売上規模、利益率、市場シェア

### 5. 主要取引先

- 有価証券報告書の「主要な販売先」を確認（売上の10%以上を占める取引先は開示義務あり）
- 特定顧客への依存度が高い場合はリスク要因として記録
- サプライチェーン上の重要な仕入先も把握

## 出力形式

```
## 事業構造

### セグメント構成
| セグメント | 売上高 | 売上比率 | 営業利益 | 利益比率 | 利益率 | PPM分類 |
|---|---|---|---|---|---|---|

### 外部環境の主要影響因子
- [因子1]: [影響の方向と程度]
- [因子2]: ...

### 競合環境
| 項目 | 対象社 | 競合A | 競合B |
|---|---|---|---|

### 主要取引先
- [取引先名]（売上比率: ○%）

### 事業構造の要約
[2〜3文で事業構造の本質を要約]
```
