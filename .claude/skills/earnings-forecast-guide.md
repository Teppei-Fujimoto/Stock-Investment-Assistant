# 決算予測マニュアル（earnings-forecaster用）

## 予測の手順

### 1. 前提条件を整理

- 前Q実績からの推移（季節性を考慮）
- 事業環境分析（business-analyst参照）
- 業績トレンド分析（fundamental-analyst参照）
- 為替前提（輸出企業の場合）
- 原材料価格の前提
- 特殊要因の有無

### 2. 自分の予測を作成

| 項目 | 自分の予測 | 根拠 |
|---|---|---|
| 売上高 | | |
| 営業利益 | | |
| 経常利益 | | |
| 当期純利益 | | |

### 3. コンセンサス予想を把握

- 証券会社のアナリストレポート、四季報予想、IFIS等のコンセンサスデータ
- 自分の予測とコンセンサスの乖離を明確にする

### 4. 上振れ・下振れシナリオ

- **上振れシナリオ**: どの前提が上振れたらどの数字がいくら改善するか
- **下振れシナリオ**: どのリスクが顕在化したらどの数字が悪化するか
- 各シナリオの発生確率を概算

---

## 決算短信の読み方（参考）

**1ページ目（優先順）**: 売上高/営業利益/経常利益/純利益（前年同期比%）、通期予想修正の有無、配当予想修正、EPS/BPS/配当

**2ページ目以降**: セグメント別業績、業績増減の要因説明、通期予想前提条件、キャッシュフロー
