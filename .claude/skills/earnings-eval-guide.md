# 決算評価マニュアル（earnings-evaluator用）

## 評価の手順

### 1. ヘッドライン数値の判定

予測・コンセンサスと実績を比較:

| 項目 | 予測 | コンセンサス | 実績 | vs予測 | vs市場 | 判定 |
|---|---|---|---|---|---|---|
| 売上高 | | | | | | 〇/△/✕ |
| 営業利益 | | | | | | 〇/△/✕ |
| 経常利益 | | | | | | 〇/△/✕ |
| 当期純利益 | | | | | | 〇/△/✕ |

判定基準: 〇=予想比+5%以上 / △=±5%以内 / ✕=予想比-5%以上

### 2. 実績の質の判定

上振れ/下振れが以下のどれか:
- **構造的要因**: 事業モデルの改善、新製品寄与 → 持続的
- **一過性要因**: 為替差益、資産売却益、補助金 → 一時的
- **会計要因**: 連結範囲変更、会計基準変更 → 見かけ上の変化

### 3. ガイダンス（通期予想）の評価

| ガイダンス変更 | 評価ポイント |
|---|---|
| 上方修正 | 修正幅は十分か？保守的でさらなる上方修正余地があるか？ |
| 据え置き | 進捗率から見て妥当か？本来修正すべきなのに据え置いていないか？ |
| 下方修正 | 一過性か構造的か？修正後の水準は底か？ |

#### 進捗率の目安（3月期決算企業）
| 時点 | 標準的な進捗率 |
|---|---|
| 1Q（4-6月） | 20〜25% |
| 2Q（4-9月） | 45〜50% |
| 3Q（4-12月） | 70〜75% |

進捗率が標準を大きく上回っている場合、上方修正の可能性が高い。

### 4. セグメント分析

- 各セグメントの増減要因を確認
- 成長ドライバーのセグメントが順調に伸びているか
- 問題児セグメントが悪化していないか

### 5. 投資判断への影響

- 投資適格基準（A/B/C）への影響: 基準を満たす方向に進んでいるか、外れているか
- アクション推奨: 買い増し/保持/一部利確/撤退
