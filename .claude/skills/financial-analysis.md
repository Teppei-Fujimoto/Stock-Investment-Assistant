# 財務分析マニュアル

## 分析の目的

売上・利益の推移から業績変化の兆しを検出し、その変化が構造的か一過性かを判定する。
また、FCF（フリーキャッシュフロー）分析により企業のキャッシュ創出力を評価する。

## 四半期業績の分析手順

### 1. トップライン（売上高）の分析

- **YoY（前年同期比）**: 季節性を排除した成長率。最重要指標
- **QoQ（前四半期比）**: 直近のモメンタム。ただし季節性に注意
- 可能であれば売上を **数量 × 単価** に分解し、成長ドライバーを特定
- 新規受注高（受注残）がある業種では受注動向も確認

### 2. 利益構造の分析

#### 利益の段階
| 利益段階 | 見るべきポイント |
|---|---|
| 売上総利益（粗利） | 原価構造の変化。原材料費、外注費の影響 |
| 営業利益 | 本業の収益力。販管費の効率性 |
| 経常利益 | 営業外損益（金融収支、為替差損益）を含む。日本では重視される |
| 当期純利益 | 特別損益を含む最終利益。一過性要因の影響を受けやすい |

#### マージン分析
- 売上総利益率（粗利率）の推移
- 営業利益率の推移
- マージンが改善している場合、その要因は:
  - 売上増による固定費レバレッジ → 持続的
  - 原材料費低下 → 一過性の可能性
  - コスト削減施策 → 持続性を要確認
  - プロダクトミックスの変化 → 構造的変化の可能性

### 3. 業績変化の兆し検出パターン

以下のシグナルに注目する:

**ポジティブシグナル**:
- 売上成長率が加速（前Q: +15% → 今Q: +25%）
- 営業利益率が複数四半期連続で改善
- 受注残高が増加トレンド
- セグメント別で新しい成長ドライバーが出現
- ガイダンス上方修正

**ネガティブシグナル / 要注意シグナル**:
- 売上は伸びているが利益が伸びない（マージン悪化）
- 売掛金の伸びが売上の伸びを大幅に上回る
- 棚卸資産の異常な増加
- 一過性の利益（固定資産売却益、補助金）で嵩上げ
- ガイダンス据え置き（本来上方修正すべき進捗率なのに）

### 4. FCF（フリーキャッシュフロー）分析

#### 計算方法
```
FCF = 営業キャッシュフロー - 設備投資関連支出
```

**重要**: 投資キャッシュフローから以下の非設備投資項目を **排除** する:
- 有価証券の取得・売却（投資有価証券、関係会社株式）
- 定期預金の増減
- 貸付金の増減
- 事業譲受・M&A関連支出（ケースバイケースで判断）

実務的には「有形固定資産の取得による支出」+「無形固定資産の取得による支出」を設備投資として使う。

#### FCF利回りの計算
```
FCF利回り = FCF ÷ 時価総額 × 100（%）
```

#### 評価基準
- FCF利回りを同業他社と比較し、相対的な位置を把握
- FCFの推移（3〜5年）を確認し、安定性を評価
- 高いFCF利回り + 高い成長率 = バリューグロース投資（B）の有力候補

## 会計基準の違い

### IFRS vs 日本基準の主な相違点
| 項目 | IFRS | 日本基準 |
|---|---|---|
| のれん | 非償却（減損テストのみ） | 20年以内で定額償却 |
| 営業利益の定義 | 規定なし（各社定義） | 売上総利益−販管費 |
| 開発費 | 要件を満たせば資産計上 | 原則費用処理 |
| リース | 原則オンバランス（IFRS16） | ファイナンスリースのみ |

**注意**: IFRS採用企業の営業利益は「事業利益」「コア営業利益」など独自定義の場合がある。比較時は定義を揃えること。
