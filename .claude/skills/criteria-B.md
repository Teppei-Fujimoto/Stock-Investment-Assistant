# 投資基準B: バリューグロース投資

**コンセプト**: 高い成長率でストック型成長を実現しているが、割安な銘柄を捉える

## 定性的評価（成長の質）

- **閾値**: 売上・ARR・営業利益等が **約30%以上の水準で成長**
- **ストック型成長**: SaaS/サブスクリプション・リカーリングレベニュー・長期契約ベース
- 解約率（チャーンレート）が低いこと
- NRR（ネットレベニューリテンション）が100%以上

## 定量的評価（バリュエーション要件）

- **PEG ≦ 1.0 が最低ライン**（PEG < 0.5 は非常に魅力的）
  - 使用する成長率: 会社予想/コンセンサス/自分の予想の中で最も信頼性の高いもの
- **FCF利回り**: 投資CFから証券売買等の非設備投資項目を排除して計算。同業他社比較で優位であること
- FCF利回り高 + 成長率高 = 理想的なバリューグロース銘柄
