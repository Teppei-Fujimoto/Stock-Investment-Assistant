# モメンタム分析マニュアル

## 分析の目的

テーマ性・価格トレンド・出来高・機関投資家動向から、対象銘柄がモメンタムに乗れるかを判定する。
主にC（モメンタム投資）で使用するが、A（カタリストの市場認知度）やB（成長テーマの追い風）の判断にも貢献する。

## テーマ・トレンド分析

### 1. テーマの特定と分類

現在の日本株市場で注目されるテーマカテゴリ:
- **国策テーマ**: 防衛、半導体・経済安全保障、脱炭素・GX、デジタル田園都市
- **技術テーマ**: AI・生成AI、量子コンピュータ、宇宙、ロボティクス
- **社会テーマ**: インバウンド、人手不足・省力化、介護・ヘルスケア
- **金融テーマ**: 金利上昇（銀行・保険）、円安（輸出）、東証改革（低PBR）

### 2. テーマの持続性評価

| 持続性レベル | 特徴 | 例 |
|---|---|---|
| 数年以上 | 政策・法制度に裏付け、構造的変化 | 防衛費増額、半導体国産化 |
| 1年程度 | 明確なイベントドライバー | 万博、選挙サイクル |
| 数ヶ月 | 話題性主導、裏付け弱い | SNSバズ銘柄、短期的ブーム |
| 数週間以下 | 投機的、ファンダメンタルズ不在 | 仕手筋、材料出尽くしリスク大 |

**投資基準C**: テーマの持続期間が **数ヶ月以上** と見込めること

### 3. テーマ内での銘柄のポジション

- **本命銘柄**: テーマの恩恵を最も直接的に受ける。業績へのインパクトが大きい
- **関連銘柄**: 間接的な恩恵。連想買いで上がるが持続性に欠ける場合が多い
- **出遅れ銘柄**: テーマに関連するが株価がまだ反応していない

## 価格トレンド分析

### 移動平均線
- **5日移動平均**: 超短期トレンド
- **25日移動平均**: 短期トレンド
- **75日移動平均**: 中期トレンド
- **200日移動平均**: 長期トレンド

**ゴールデンクロス**: 短期線が長期線を上抜け → 上昇トレンドのシグナル
**デッドクロス**: 短期線が長期線を下抜け → 下落トレンドのシグナル

### 価格位置
- **52週高値との比較**: 52週高値更新はブレイクアウトのシグナル
- **52週安値との比較**: 安値圏にある場合はリバーサルの可能性（ただしC投資では避ける）
- **直近のレンジ**: ブレイクアウトの有無を確認

## 出来高分析

### 出来高の判定基準
- **出来高急増の定義**: 直近20日平均の **3倍以上**
- 株価上昇 + 出来高増加 = トレンド継続の確認
- 株価上昇 + 出来高減少 = トレンドの弱体化（要注意）
- 株価下落 + 出来高増加 = 売り圧力の強さ（撤退シグナル）

### 価格帯別出来高
- 過去の出来高が集中した価格帯は支持線・抵抗線として機能
- ブレイクアウト後に出来高の薄い価格帯に入ると急騰しやすい

## 機関投資家動向

### 確認すべきデータ
- **大量保有報告書（5%ルール）**: 発行済株式の5%以上を取得した場合に開示
- **空売り残高**: 空売り残の増減。空売り残減少は買い戻し圧力
- **投資部門別売買動向**: 海外投資家の買い越し・売り越し（セクター別にも確認）
- **信用取引残高**: 信用買い残の積み上がりは将来の売り圧力

### 機関投資家の買い集め兆候
- 出来高が増加しつつ、株価が底堅い（下がらない）
- 大量保有報告で著名機関が新規取得
- 空売り残が減少トレンド

## 日本株特有の季節性パターン

- **1月効果**: 年初は小型株が強い傾向
- **節分天井・彼岸底**: 2月高値、3月安値のアノマリー（信頼度は低い）
- **5月**: "Sell in May"。海外投資家の利益確定売り
- **年末ラリー**: 11-12月の上昇傾向。新NISA枠の買い需要
- **配当権利取り**: 3月末・9月末の権利確定前に買いが入りやすい
