# 銘柄スクリーニングマニュアル

## 分析の目的

株探（Kabutan）やSNS（X/YouTube）から投資候補を発掘し、投資適格基準A/B/Cへの適合可能性を一次評価する。

---

## 株探（kabutan.jp）の活用

### 主要チェックポイント

#### 本日のイチオシ銘柄
- 株探トップページの「イチオシ銘柄」セクション
- 注目理由（テクニカル/ファンダメンタル/テーマ）を確認
- 株探のイチオシは広告銘柄ではないかも注意する

#### 決算速報
- 決算発表シーズンには「決算速報」セクションを優先チェック
- 上方修正・下方修正の一覧
- サプライズ決算（予想比大幅上振れ/下振れ）

#### 株価注意報
- ストップ高/ストップ安銘柄
- 出来高急増銘柄
- 年初来高値/安値更新銘柄

### 株探データの読み方

#### 業績ページ
- 四半期業績の推移
- 会社予想とコンセンサスの比較
- 過去の上方修正/下方修正パターン

#### 財務ページ
- 自己資本比率
- 有利子負債比率
- ROE/ROA

#### チャートページ
- 日足/週足/月足のトレンド
- 出来高の推移
- 移動平均線との位置関係

## SNS情報の活用

### X（Twitter）

#### 情報収集のポイント
- 日本株投資家コミュニティのキーワード: #日本株, #株式投資, #決算
- 銘柄コードでの検索（例: $7203, #7203）
- 決算発表直後のリアルタイム反応

#### 信頼度評価基準

| 信頼度 | 特徴 |
|---|---|
| **高** | 過去の実績が追跡可能、具体的な数字と根拠を提示、ポジション（保有/非保有）を明示 |
| **中** | 一定のフォロワー数、ある程度の分析は含むが根拠が薄い |
| **低** | 匿名・実績不明、「爆上げ」等の煽り表現、特定銘柄の一方的な推奨 |

#### 要注意パターン
- 突然の大量ツイート（仕手筋の可能性）
- 小型株への集中的な推奨（ポンプ&ダンプの可能性）
- 「●●倍になる」等の極端な予測
- フォロワー購入の疑い（フォロワー数に対してエンゲージメントが極端に低い）

### YouTube

#### 情報収集のポイント
- 投資系YouTuberの銘柄紹介
- 決算解説動画
- 市場展望・テーマ解説

#### 信頼度評価基準
- 過去の推奨銘柄のパフォーマンスが追跡可能か
- 分析の深さ（数字を使った議論か、印象論か）
- 広告案件の有無（「PR」表記があるか）

## 一次スクリーニングの通過基準

発見した銘柄を以下の基準で一次選別する:

### 必須条件（全投資タイプ共通）
- [ ] 時価総額50億円以上（流動性確保のため）
- [ ] 出来高が一定水準以上（日次10万株以上が目安）
- [ ] 直近で不祥事・行政処分がない

### A（カタリスト投資）への適合可能性
- [ ] 直近の業績に+30%以上の変化が見られる（または兆し）
- [ ] 変化の背景が構造的と思われる
- [ ] まだ市場で広く注目されていない（初動に近い）

### B（バリューグロース投資）への適合可能性
- [ ] 売上またはARRが30%以上成長
- [ ] ストック型のビジネスモデル（SaaS、サブスクリプション等）
- [ ] PERが過剰に高くない印象

### C（モメンタム投資）への適合可能性
- [ ] 明確なテーマに関連
- [ ] 出来高が増加トレンド
- [ ] 株価がブレイクアウトしている、または直前

## 出力形式

```
## 本日の注目銘柄

### [順位]. [企業名]（[証券コード]）
- **ソース**: [株探イチオシ/X/YouTube/決算速報]
- **セクター**: [業種]
- **注目理由**: [1〜2文]
- **A/B/C適合性**:
  - A（カタリスト）: [可能性あり/低い] — [理由]
  - B（バリューグロース）: [可能性あり/低い] — [理由]
  - C（モメンタム）: [可能性あり/低い] — [理由]
- **推奨アクション**: [要分析/見送り]

### ソース信頼度
- [ソース名]: [高/中/低] — [判定理由]
```
