# バリュエーション手法マニュアル

## 分析の目的

対象企業の株価が成長率・収益力に対して割安か割高かを判定する。
全ての投資タイプ（A/B/C）でバリュエーション評価が必要。

## 主要バリュエーション指標

### PER（株価収益率）

```
PER = 株価 ÷ 1株当たり予想純利益（EPS）
```

- **予想PER（フォワードPER）** を使用する（実績PERではなく）
- 日本株の平均PERは概ね14〜16倍（米国株より低い傾向）
- 判定基準:
  - セクター中央値との比較が最重要
  - 過去5年のPERレンジとの比較
  - 成長率が高ければ高PERも許容される → PEGで判断

### PEG（株価収益成長率）

```
PEG = PER ÷ 予想利益成長率（%）
```

- **投資基準Bの最重要指標**: PEG ≦ 1.0 が最低ライン、低いほど良い
- 利益成長率は、会社予想・コンセンサス予想・自分の予想のうち最も信頼性の高いものを使用
- PEG < 0.5: 非常に割安（成長率が本当に持続するか要確認）
- PEG 0.5〜1.0: 割安
- PEG 1.0〜1.5: 適正
- PEG > 1.5: 割高

### PBR（株価純資産倍率）

```
PBR = 株価 ÷ 1株当たり純資産（BPS）
```

- PBR < 1.0 は「解散価値割れ」。TSEのPBR改善要請により注目度が高い
- PBR単体では判断せず、ROEとセットで評価:
  - 高ROE + 低PBR = 割安（投資候補）
  - 低ROE + 低PBR = バリュートラップの可能性
- 含み資産（不動産、投資有価証券）を持つ企業は実態PBRが低い可能性

### PSR（株価売上高倍率）

```
PSR = 時価総額 ÷ 売上高
```

- 赤字企業やSaaS企業など、利益が出ていない成長企業の評価に使用
- セクター・成長ステージによる差が大きいため、必ず同業比較で使う
- SaaS企業: ARR成長率30%以上なら PSR 10〜15倍は許容範囲

### EV/EBITDA

```
EV = 時価総額 + 有利子負債 − 現金同等物
EBITDA = 営業利益 + 減価償却費
EV/EBITDA = EV ÷ EBITDA
```

- 資本構成や減価償却の違いを排除した比較が可能
- 設備投資が大きい業種（製造業、通信、不動産）で有用
- 日本株平均: 8〜10倍程度

### RSI（相対力指数）

```
RSI = 値上がり幅の平均 ÷ (値上がり幅の平均 + 値下がり幅の平均) × 100
```

- 14日RSIが標準
- RSI > 70: 過熱圏（カタリスト投資Aのバリュエーション要件で要チェック）
- RSI < 30: 売られすぎ
- RSI単体ではなく、他の指標と組み合わせて使用

## バリュートラップの回避

以下のケースは「見た目は割安だが実際は危険」な可能性がある:
- 低PER + 利益減少トレンド（来期のPERは高くなる可能性）
- 低PBR + 低ROE + 改善の兆しなし
- 高配当利回り + 業績悪化（減配リスク）
- 特別利益で一時的にEPSが嵩上げされている

## 投資タイプ別のバリュエーション要件

### A（カタリスト投資）
- 将来利益をバリュエーションが **織り込んでいない** こと
- RSIが過熱圏（70超）でないこと
- 業績変化を織り込む前のPERが、変化後の利益水準で見ると割安であること

### B（バリューグロース投資）
- PEG ≦ 1.0（最低ライン）
- FCF利回りが同業他社と比較して優位
- ストック型成長が継続する前提で、現在の株価が割安であること

### C（モメンタム投資）
- テーマの初動〜中盤であり、バリュエーションが極端に割高でないこと
- RSIやモメンタム指標で過熱度を確認
