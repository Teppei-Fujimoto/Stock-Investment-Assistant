# 投資適格基準

## 概要

ユーザーは3つの投資タイプを使い分けている。各タイプには定性的評価と定量的評価（バリュエーション）の両方が必要。

---

## A: カタリスト投資

**コンセプト**: 将来的に拡大しそうな業績変化のはじまりを捉える

### 定性的評価（業績変化の検出）

#### 対象指標
- 売上高（数量・単価に分解して確認）
- 新規受注高
- 利益率（売上総利益率、営業利益率）
- 営業利益

#### 変化の閾値
- 上記指標に **+30〜40%以上の変化** が出ている、もしくは出る兆しがある

#### 持続性の判定
- その変化が **中長期的に継続** すると判断できること
- 持続的な変化の例:
  - 新製品・新サービスの立ち上がり
  - 市場拡大（TAMの成長）
  - 構造的なコスト改善（DX、自動化）
  - 規制変更による追い風
- 一過性の変化（不適格）の例:
  - 為替の一時的な追い風
  - 補助金・助成金の一時収入
  - 前年の落ち込みからの反動増
  - 一度限りの大型受注

#### 加速の評価
- ポジティブな変化が逆流しないのは当然として、さらに **加速** すると考えられる場合は高評価
- 加速のシグナル:
  - 成長率が四半期ごとに上昇
  - 新規顧客獲得ペースの加速
  - クロスセル/アップセルの進展
  - 新市場への展開

### 定量的評価（バリュエーション要件）

- その変化によって高まりうる **将来利益をバリュエーションが織り込んでいない** こと
  - 現在のPERは過去の利益水準で評価されており、変化後の利益で計算すると割安になるケース
  - アナリストカバレッジが少なく、市場の認知度が低い
- **RSI的に加熱しすぎていない** こと
  - 14日RSIが70を大幅に超えている場合はエントリータイミングを再考

---

## B: バリューグロース投資

**コンセプト**: 高い成長率でストック型成長を実現しているが、割安な銘柄を捉える

### 定性的評価（成長の質）

#### 成長の閾値
- 売上、ARR（年間経常収益）、営業利益などが **約30%以上の水準で成長**

#### ストック型成長の要件
- 収益がストック型（積み上がり型）であること
  - SaaS/サブスクリプションモデル
  - リカーリングレベニュー（保守・メンテナンス収入）
  - 長期契約ベースの収益
- 解約率（チャーンレート）が低いこと
- NRR（ネットレベニューリテンション）が100%以上であること

### 定量的評価（バリュエーション要件）

#### PEGレシオ
- **PEG ≦ 1.0 が最低ライン**
- 低いほど良い。PEG < 0.5 は非常に魅力的
- 使用する成長率: 会社予想/コンセンサス/自分の予想の中で最も信頼性の高いもの

#### FCF利回り
- フリーキャッシュフロー利回りを算出
- **投資CFから証券売買等の非設備投資項目は排除** して計算
- 同業他社と比較して優位であること
- FCF利回りが高い + 成長率が高い = 理想的なバリューグロース銘柄

---

## C: モメンタム投資

**コンセプト**: 業界全体に対するモメンタムに順張りする

### テーマ性の要件
- **国策テーマ・市場の大きなテーマに合致** していること
- 対象企業がそのテーマの **本命銘柄** であること（関連銘柄・出遅れ銘柄は慎重に）
- テーマの将来的な持続期間が **数ヶ月以上** と見込めること

### テクニカル要件
- **出来高が直近20日平均の3倍以上に急増**
- **52週高値を更新**、または **直近のレンジをブレイクアウト**
- 移動平均線が上昇配列（短期 > 中期 > 長期）

### 機関投資家動向
- **機関投資家の買い集め兆候**:
  - 大量保有報告書での新規取得
  - 空売り残高の減少
  - 外国人投資家の買い越し

---

## 投資タイプの判定フロー

```
1. 対象銘柄に業績変化の兆候がある？
   ├─ Yes → 変化は+30%以上で持続的？
   │         ├─ Yes → バリュエーションが未織り込み？
   │         │         ├─ Yes → A（カタリスト投資）候補
   │         │         └─ No → タイミング待ち
   │         └─ No → 一過性の変化、見送り
   └─ No → 次へ

2. ストック型の高成長（30%以上）を実現？
   ├─ Yes → PEG ≦ 1.0？
   │         ├─ Yes → FCF利回りも良好？
   │         │         ├─ Yes → B（バリューグロース投資）候補
   │         │         └─ No → B候補だが要注意
   │         └─ No → 割高、見送りまたはタイミング待ち
   └─ No → 次へ

3. 強いテーマ性とモメンタムがある？
   ├─ Yes → テーマ持続数ヶ月以上 + テクニカル条件充足？
   │         ├─ Yes → C（モメンタム投資）候補
   │         └─ No → 短期投機的、見送り
   └─ No → 現時点では投資対象外
```

## 複数タイプへの該当

- 1銘柄が複数の投資タイプに該当する場合がある（例: AとBの両方）
- その場合は最も強く該当するタイプをメイン、他をサブとして報告する
- 複数タイプに該当する銘柄は確信度が高いと判断できる
