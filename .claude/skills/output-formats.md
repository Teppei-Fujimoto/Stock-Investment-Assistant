# 出力フォーマット集

## 共通ルール

- 全て日本語で出力
- 専門的な略語は **初回使用時のみ** 括弧内で簡単に解説する（例: OP（Operating Profit＝営業利益）、PEG（Price/Earnings to Growth＝株価収益成長率）、FCF（Free Cash Flow＝フリーキャッシュフロー））。2回目以降は略語のみで可
- 証券コードは常に明記（例: トヨタ自動車（7203））
- 通貨は「円」表記
- 決算期は日本慣行（例: 2025年3月期、FY2024）
- 判定には 〇/△/✕ を使用
- 数値は桁区切りあり（例: 1,234百万円）
- 出所が明確なデータは出所を添える
