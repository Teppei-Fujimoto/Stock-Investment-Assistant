# 決算評価マニュアル

## 分析の目的

### 決算予測（earnings-forecaster用）
分析部門の情報をもとに次Qの決算を予測し、外部コンセンサスとの乖離を整理する。

### 決算評価（earnings-evaluator用）
発表された決算を予測・コンセンサスと比較し、実績の質を判定して投資判断への影響を評価する。

---

## 決算予測の手順

### 1. 予測の前提条件を整理

- 前Q実績からの推移（季節性を考慮）
- business-analystの事業環境分析
- fundamental-analystの業績トレンド分析
- 為替前提（輸出企業の場合）
- 原材料価格の前提
- 特殊要因の有無

### 2. 自分の予測を作成

| 項目 | 自分の予測 | 根拠 |
|---|---|---|
| 売上高 | | |
| 営業利益 | | |
| 経常利益 | | |
| 当期純利益 | | |

### 3. コンセンサス予想を把握

- 証券会社のアナリストレポート
- 四季報予想
- コンセンサスデータ（IFIS等）
- 自分の予測とコンセンサスの乖離を明確にする

### 4. 上振れ・下振れシナリオ

- **上振れシナリオ**: どの前提が上振れたらどの数字がいくら改善するか
- **下振れシナリオ**: どのリスクが顕在化したらどの数字が悪化するか
- 各シナリオの発生確率を概算

---

## 決算短信の読み方

### 1ページ目（サマリー）で確認すべき項目（優先順）

1. **売上高・営業利益・経常利益・当期純利益**（前年同期比%）
2. **通期業績予想の修正** の有無（上方修正/下方修正/据え置き）
3. **配当予想の修正** の有無
4. **1株当たり情報**: EPS、BPS、配当

### 2ページ目以降で確認すべき項目

5. **セグメント別業績**: どのセグメントが牽引/足を引っ張っているか
6. **定性的な業績説明**: 増減の要因説明
7. **通期予想の前提条件**: 為替、原材料、その他の前提
8. **キャッシュフロー**: 営業CF、投資CF、財務CF

---

## 決算評価の手順

### 1. ヘッドライン数値の判定

予測・コンセンサスと実績を比較:

| 項目 | 予測 | コンセンサス | 実績 | vs予測 | vs市場 | 判定 |
|---|---|---|---|---|---|---|
| 売上高 | | | | | | 〇/△/✕ |
| 営業利益 | | | | | | 〇/△/✕ |
| 経常利益 | | | | | | 〇/△/✕ |
| 当期純利益 | | | | | | 〇/△/✕ |

判定基準:
- 〇: 予想比+5%以上の上振れ
- △: 予想比±5%以内（ほぼ予想通り）
- ✕: 予想比-5%以上の下振れ

### 2. 実績の質の判定

上振れ/下振れが以下のどれに該当するか:
- **構造的要因**: 事業モデルの改善、新製品寄与 → 持続的
- **一過性要因**: 為替差益、資産売却益、補助金 → 一時的
- **会計要因**: 連結範囲変更、会計基準変更 → 数字の見かけ上の変化

### 3. ガイダンス（通期予想）の評価

| ガイダンス変更 | 評価ポイント |
|---|---|
| 上方修正 | 修正幅は十分か？保守的でさらなる上方修正余地があるか？ |
| 据え置き | 進捗率から見て据え置きは妥当か？本来修正すべきなのに据え置いていないか？ |
| 下方修正 | 一過性か構造的か？下方修正後の水準は底か？ |

#### 進捗率の目安（3月期決算企業の場合）
| 時点 | 標準的な進捗率 |
|---|---|
| 1Q（4-6月） | 20〜25% |
| 2Q（4-9月） | 45〜50% |
| 3Q（4-12月） | 70〜75% |

進捗率が標準を大きく上回っている場合、上方修正の可能性が高い。

### 4. セグメント分析

- 各セグメントの増減要因を確認
- 成長ドライバーのセグメントが順調に伸びているか
- 問題児セグメントが悪化していないか

### 5. 投資判断への影響

決算結果を踏まえ、以下を判定:
- 投資適格基準（A/B/C）への影響: 基準を満たす方向に進んでいるか、外れているか
- アクション推奨: 買い増し/保持/一部利確/撤退
