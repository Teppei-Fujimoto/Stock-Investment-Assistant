# 投資基準A: カタリスト投資

**コンセプト**: 将来的に拡大しそうな業績変化のはじまりを捉える

## 定性的評価（業績変化の検出）

- 対象指標: 売上高・新規受注高・利益率・営業利益
- **閾値**: 上記指標に **+30〜40%以上の変化** が出ている、もしくは出る兆しがある
- **持続性**: 変化が中長期的に継続すること（新製品立ち上がり、市場拡大、構造的コスト改善、規制追い風）
- **除外**: 一過性の変化（為替一時追い風、補助金、前年反動増、一度限りの大型受注）
- **加速**: さらに加速すると考えられる場合は高評価（成長率上昇、新規顧客獲得加速、クロスセル進展）

## 定量的評価（バリュエーション要件）

- 変化後の将来利益をバリュエーションが **まだ織り込んでいない** こと（現PERは変化前の利益水準で評価されている）
- **RSI < 70**: 14日RSIが70を大幅に超えている場合はエントリータイミングを再考
